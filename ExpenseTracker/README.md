# ShearSpend - Expense Tracker

A web-based expense tracking application built with Django.

## Features
- Add expenses
- Manage partners
- Track spending
- User authentication

## Tech Stack
- Python
- Django
- HTML/CSS
- Bootstrap

## Installation

Clone the repository

git clone https://github.com/Nasir-Mureed/ShearSpend.git

Go to project folder

cd ShearSpend

Create virtual environment

python -m venv venv
source venv/bin/activate

Install dependencies

pip install -r requirements.txt

Run migrations

python manage.py migrate

Run server

python manage.py runserver
